# Code Of Conduct

This project should be practical, careful, and welcoming.

Expected behavior:

- be respectful
- assume users care about their libraries
- avoid shaming messy collections
- prioritize safety over cleverness
- explain technical concepts clearly

Unacceptable behavior:

- harassment
- discriminatory language
- sharing private user library data
- encouraging unsafe commands without context

Maintainers may remove content or restrict participation to protect the project and its users.
